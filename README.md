
# HR Data Analysis Project

This project explores HR hiring patterns, joining behavior, and predictive modeling for employee offer acceptance.

---

## Key Model Performance
- **Accuracy:** 0.81  
- **Precision:** 0.89  
- **Recall (Sensitivity):** 0.72  
- **Specificity:** 0.91
